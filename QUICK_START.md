# 🚀 Быстрый старт

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите лаунчер:**
   ```bash
   python main_launcher.py
   ```
   
   Или дважды кликните на `run_launcher.bat`

## 🎯 Доступные боты

### 💊 Авто Лечение
- **Файл:** `sme/hint.py`
- **Описание:** Распознает изображения и показывает подсказки
- **Управление:** Caps Lock для включения/выключения

### 🏗️ Авто Стройка  
- **Файл:** `build/build/auto_keypress.py`
- **Описание:** Автоматически нажимает кнопки E, F, H
- **Управление:** Caps Lock для включения/выключения

### 🍳 Авто Готовка
- **Файл:** `cock/main.py`
- **Описание:** Автоматизирует готовку с выбором рецептов
- **Управление:** Caps Lock для включения/выключения

### 🚕 Авто Такси
- **Файл:** `tax/find_click.py`
- **Описание:** Автоматически ловит заказы такси
- **Управление:** Caps Lock для включения/выключения

### 🐄 Авто Доение
- **Файл:** `farm/cow_milking.py`
- **Описание:** Автоматическое доение коровы на ферме
- **Управление:** Caps Lock для включения/выключения

## ⚡ Горячие клавиши

- **Caps Lock** - включение/выключение бота
- **Q** или **Ctrl+Q** - выход из бота
- **Ctrl+C** - принудительная остановка (в консоли)

## 🔧 Настройка

Каждый бот можно настроить под свои нужды, изменив параметры в соответствующих файлах:
- Пороги уверенности для распознавания
- Задержки между действиями  
- Области поиска на экране
- Последовательности нажатий клавиш
