# 🎮 Игровой Бот Лаунчер

Современный GUI лаунчер для запуска игровых ботов автоматизации.

## 🚀 Возможности

- **💊 Авто Лечение** - Распознавание изображений и подсказки для лечения
- **🏗️ Авто Стройка** - Автоматическое нажатие кнопок на стройке
- **🍳 Авто Готовка** - Автоматизация готовки с выбором рецептов
- **🚕 Авто Такси** - Автоматическая ловля заказов такси
- **🐄 Авто Доение** - Автоматическое доение коровы на ферме

## 📦 Установка

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск лаунчера
```bash
python main_launcher.py
```

## 🎯 Как использовать

1. **Запустите лаунчер**: `python main_launcher.py`
2. **Выберите нужного бота** из списка
3. **Нажмите "🚀 Запустить"** для запуска выбранного бота
4. **Управление ботами**: 
   - Включение: **Caps Lock**
   - Выключение: **Caps Lock** (повторно)
   - Выход из бота: **Q** или **Ctrl+Q**

## ⚙️ Настройка

### Для бота готовки:
- Поместите изображения ингредиентов в папку `cock/`
- Настройте порог уверенности в `confidence_threshold`
- Отрегулируйте задержки между действиями

### Для бота лечения:
- Добавьте свои шаблоны изображений в папку `sme/`
- Настройте `TEMPLATE_ACTIONS` в `hint.py`
- Отрегулируйте `TEMPLATE_THRESHOLD`

### Для бота строительства:
- Настройте последовательность нажатий в `auto_keypress.py`
- Отрегулируйте задержки между действиями

### Для бота такси:
- Настройте область поиска в `find_click.py`
- Добавьте свои изображения для поиска заказов
- Отрегулируйте координаты и размеры областей

### Для бота доения коровы:
- Настройте скорость доения в `CONFIG["milking_speed"]`
- Отрегулируйте задержки между нажатиями клавиш
- Измените клавиши доения в `CONFIG["keys_to_press"]`

## 🔧 Требования

- **Python**: 3.8+
- **ОС**: Windows 10/11
- **Библиотеки**: PyQt6, OpenCV, PyAutoGUI, NumPy

## ⚠️ Важные замечания

1. **Анти-чит**: Используйте `pydirectinput` для обхода систем защиты
2. **Разрешение экрана**: Учитывайте масштабирование DPI в Windows
3. **Безопасность**: Запускайте только в доверенных играх
4. **Производительность**: Настройте частоту обновления для оптимизации

## 🆘 Устранение неполадок

### Ошибка "Module not found":
```bash
pip install -r requirements.txt
```

### Бот не запускается:
- Проверьте путь к файлу в лаунчере
- Убедитесь, что все зависимости установлены
- Проверьте права доступа к файлам

### Проблемы с изображениями:
- Убедитесь, что изображения находятся в правильных папках
- Проверьте формат файлов (PNG рекомендуется)
- Настройте порог уверенности для распознавания
